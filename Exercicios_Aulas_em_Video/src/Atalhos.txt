VS Code - Java
Ctrl + . -> abre opções para gerar getters, setters constructors

Terminal
cd pasta -> abre o diretorio especificado por "pasta"
cd .. -> volta ao diretório anterior
ls -> lista arquivos
clear -> limpa terminal
mkdir nome -> cria nova pasta

Git/GitHub
git add file -> adiciona "file" ao staging area
git add . -> adicona tudo a staging area
git commit -m "mensagem" -> faz commit com mensagem
git init -> começa um repositório
git status -> mostra o que foi alterado
git push origin nome_da_branch -> envia para o GitHub
git pull -> puxa do GitHub
git clone link -> copia um projeto do GitHub
-------------------------------------------------
Objetos são instâncias de uma classe.
